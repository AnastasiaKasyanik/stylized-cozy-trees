# What does it contain?
An Unreal Engine 5 project showcasing a sample of a few stylized trees I made. The project contains 3D meshes of trees, their trunk and foliage materials, as well as a global wind manager tool which is controlled via a Material Parameter Collection (MPC_GlobalWind).
